Information for RFM12 wireless module at HR25
Internal version inside HR25

NOTE: RFM12B does not need external pull up resistor (100k) on FSK/DATA/nFFS pin. RFM12 needs this pull up.

Links RFM12B:
http://www.hoperf.com/pdf/RFM12B.pdf
http://www.hoperf.com/pdf/RF12B.pdf
http://www.hoperf.com/pdf/RF12B_code.pdf

Links RFM12 (old version):
http://www.hoperf.com/pdf/RFM12.pdf
http://www.hoperf.com/pdf/RF12.pdf
http://www.hoperf.com/pdf/RF12_code.pdf

Note: JTAG debugging is possible, but RFM module will not work while JTAG is enabled. Disable JTAG for full RFM functionality.
